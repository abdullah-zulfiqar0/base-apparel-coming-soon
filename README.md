# Frontend Mentor - Base Apparel coming soon page solution

This is my solution to the **Base Apparel coming soon page** challenge on Frontend Mentor.  
This project helped me practice **HTML, CSS (Flexbox & Responsive Design), and JavaScript form validation**.

---

## Overview

### The challenge

Users should be able to:

- View the optimal layout depending on their device screen size (desktop & mobile)
- See hover states on the submit button
- Get an error message when:
  - The email field is empty
  - The email format is invalid
